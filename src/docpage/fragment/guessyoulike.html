北京推荐位和点击上报

使用场景：页面的猜你喜欢楼层

跳转点击：由于用户跳转点击时，根据网络情况会有一定的丢失。所以采取了先缓存然后在下一页面上报的方式。

页面跨域存储：现在只有wqs.jd.com和wq.jd.com两个域名的代理页面。所以猜你喜欢的楼层页面url需要在这两个域名，跳转到商详会把缓存点击数据进行上报。

曝光正确调用方式：

wa('guessyouLike',{
    action: 0, // 代表曝光
    扩展1: '111',
    扩展2：'222',
    ......
});

点击上报正确调用方式：

wa('guessyouLike',{
    action: 1, // 代表点击
    扩展1: '111',
    扩展2：'222',
    ......
});
